Description

Allows you send/receive SMS's using the twilio, smsglobal, smsgateway.ca gateways

Individual gateway features
SMS GatewaySend SMSDelivary ReceiptsReceive SMS
twilioOOO
smsglobalOXX
smsgateway.caOXX

Features

  • Configure multiple accounts for each gateway
  • View SMS history
  • Simple sending of smses for leads and partners
  • Automaticlly creates new leads if a lead or partner doesn't have that mobile number
  • Built modularly so you can easily add a new gateway of your choice

Using the module

Initial Configuration

1. Create a twilio/smsglobal/smsgateway account and add credit
2. Go to Settings->SMS Gateway Configuration->Accounts and enter the all the api settings

Sending an SMS to partners / customers / suppliers and leads

1. Simply select Send SMS from the more menu and type in your message

Sending SMS from any model

1. Add widget="smsfield" to any field you wish treat as a mobile number
2. Click on the field to open the SMS compose window

View SMS Histroy

1. Go to Settings->SMS Gateway Configuration->SMS History

Create SMS Template

1. Go to Settings->SMS Gateway Configuration->SMS Templates
2. Select the target model, sms account and type away
